Preventorium
==========
Проект, предназначенный для автоматизации работы профилактория ИГХТУ.

Описание каталога:
- Preventorium - содержит исходные коды программы.
- База данных - содержит SQL-скрипты генерации БД.
- Исходные документы - содержит документы, на основе котрых строится система.
- Модель данных - содержит модель данных, на основе которой будет построена БД.
